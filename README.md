# âš™ï¸ What Is Disruptive Innovation?

A **disruptive innovation**:

- Starts in **low-end or niche markets**.  
- Gradually improves performance and **eventually overtakes** traditional players.  
- Differs from **sustaining innovations**, which only enhance existing products.

### Typical Stages
1. Entry into a low-end or new market.  
2. Incremental improvement and expansion.  
3. Displacement of incumbent firms.
